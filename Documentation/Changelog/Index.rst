Change log
==========

Version 0.2.0
-------------

- Add extension key to composer.json file
- Add support for using toolbar with page translations

Version 0.1.1

- Fix composer.json file

Version 0.1.0
-------------

First release of Quickedit toolbar.

Features:

- Toolbar for editing page properties in page module
- New options in user settings for toolbar behavior
- Page TSconfig configuration for default page types
- Disable toolbar by user or usergroup settings
